# What is this?
This is a repository for all of my code from a university course on digital design and digital logic

# Interesting Labs?
The most complex and interesting lab in this repository is Lab 8. 
In this lab, I programmed an FPGA (DE1-Soc) to accept direct inputs from an nes controller and render a fully functional snake game with tunable speeds via switch inputs and collision detection (self-snake collisions and wall collisions)
